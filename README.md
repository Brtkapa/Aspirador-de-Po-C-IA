# Aspirador-de-Po-C-IA
Simulador de Agente Aspirador de Pó Autônomo em C. Modelagem de IA com três regimes de percepção: Ambiente Totalmente Observável e dois tipos Parcialmente Observáveis (Reflexivo e Local). O projeto demonstra profunda compreensão da Teoria de Agentes Inteligentes, modelagem de ambientes e programação procedural.
